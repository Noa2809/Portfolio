Projet Bookworm

Python NLP spaCy Project Gutenberg

Un moteur d'analyse littéraire appliqué au texte d'Alice au Pays des Merveilles : résumé automatique, extraction d'entités nommées, diversité lexicale, similarité entre livres et détection de thèmes par chapitre.

« De plus en plus curieux ! »

— Alice

Architecture du projet

L'architecture est modulaire : chaque fichier Python prend en charge un unique aspect de l'analyse. Les livres sont téléchargés automatiquement depuis Project Gutenberg et un cache local sauvegarde les résultats au format JSON pour optimiser les performances.

Python 3 spaCy Bag of Words requests Git / GitHub

Pipelines d'analyse (Modules)

loader.py

Chargement & Cache

Télécharge le texte depuis Project Gutenberg si absent et sauvegarde localement.

preprocessing.py

Nettoyage du texte

Minuscules, suppression des balises Gutenberg et retrait de la ponctuation (Regex).

lexdiv.py

Diversité Lexicale

Calcule la richesse du vocabulaire via un modèle statistique (tok, typ, hap, ttr).

summarizer.py

Résumé Extractif

Analyse et score les phrases pour extraire et générer automatiquement le meilleur résumé.
